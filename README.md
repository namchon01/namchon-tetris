# namchon-tetris

iOS 연습용 모바일 테트리스 앱 (SwiftUI)

## 실행 방법

1. Mac에서 `NamchonTetris/NamchonTetris.xcodeproj`를 Xcode로 엽니다.
2. iPhone 시뮬레이터 또는 실제 기기를 선택합니다.
3. Run (⌘R)으로 실행합니다.

## 기능 (Phase 1 MVP)

- 10×20 게임 보드
- 7종 테트로미노 (이동, 회전, 소프트/하드 드롭)
- 라인 클리어, 점수, 레벨
- 다음 블록 미리보기
- 일시정지 / 재시작 / 게임 오버
- **세로모드(Portrait) 고정**

## 화면 구성

```
┌─────────────────────┐
│  게임 보드 (10×20)   │
│                     │
├─────────────────────┤
│ 점수·레벨·다음 블록  │
│ 조작 버튼           │
│ PAUSE / RESTART     │
└─────────────────────┘
```

## 요구 사항

- iOS 17+
- Xcode 15+
